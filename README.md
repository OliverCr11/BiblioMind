# BiblioMind - Dark Luxury Book Review App

BiblioMind is a sophisticated Book Review Platform implementing seamless CRUD operations for deep literary analysis. Built using React and Tailwind CSS.

## Features
- **Visual Discovery Focus**: High-impact book covers with an elegant, gallery-like interface.
- **Dark Luxury Aesthetic**: Glassmorphism (`backdrop-blur-xl`), custom fonts, and neon purple (`#A855F7`) glowing accents.
- **Interactive UI**: Custom Flip Cards for books, hovering stars, floating background particles.
- **Complete Views**: Hero dashboard, Featured Books Grid, Review Form, Recent Community Reviews.

## Tech Stack
- Frontend: React (Vite)
- STyling: Tailwind CSS
- Icons: Lucide React

## Setup Instructions
1. Run `npm install` to install the dependencies.
2. Run `npm run dev` to start the development server.
3. Open your browser and navigate to the provided localhost URL.
